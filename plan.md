# AGI&FBHC 科研课题组官网首页 - 执行计划

## 项目概述
为 AGI&FBHC 科研课题组设计并实现一个高端、深色主题、国际化的科研官网首页。

## 阶段划分

### Stage 1: 设计阶段
- 加载 `vibecoding-webapp-swarm` 技能
- 完成 design.md 设计文档
- 确定视觉风格、配色、动画、布局

### Stage 2: 技术规划阶段
- 完成 tech-spec.md 技术规格文档
- 确定组件选型、动画实现方案

### Stage 3: 资产准备阶段
- 提取/生成需要的图片资产
- Logo 处理、背景图等

### Stage 4: 开发阶段
- 初始化 Next.js + shadcn/ui 项目
- 安装依赖
- 实现各组件：Navbar, HeroSection, ResearchGroups, PlatformsSection, UpdatesSection, AboutSection, Footer
- 构建并部署

### Stage 5: 部署阶段
- 构建生产版本
- 部署到生产环境

## 技能加载
- Stage 1: `vibecoding-webapp-swarm` (design)
- Stage 2: `vibecoding-webapp-swarm` (tech-spec)
- Stage 3: `vibecoding-webapp-swarm` (assets)
- Stage 4: `vibecoding-webapp-swarm` (development)
- Stage 5: `vibecoding-webapp-swarm` (deploy)
